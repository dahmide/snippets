# snippets

A personal collection of small code pieces, utilities, and components for quick copy-paste.  
Designed to be easy to browse on GitHub and grab code fast, especially useful for quick setups or low-power situations.

---

## 📂 Structure

Organize your snippets by language or type for faster access:

- `react/`
- `js/`
- `ts/`
- `css/`
- `python/`

## 📌 Usage

1. Browse the repo on [GitHub](https://github.com/dahmide/snippets).  
2. Click on the file you need.  
3. Copy & paste directly into your project.

## 🛠 Contribution

- Add new snippets in the correct folder.  
- Use clear filenames and add comments for clarity.  
- Optional: create mini-READMEs in each folder for detailed notes.

---

## 💡 Notes

- This repo is intentionally simple: no dependencies, minimal friction.  
- Designed to save time, especially when you’re in a hurry.  
- Use GitHub web interface for fastest copy-paste experience.
